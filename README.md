# CorpsSquadSpectatorPortal

Open the submitted zip file, and then once you have navigated to the parent folder (and are on
EECSnet), in your terminal, run:

  docker compose up –build

To open the frontend in your browser:

  http://localhost:3000

To login:

  Username: admin
  Password: iamabigfroghater

To view the backend:

  http://localhost:5001/events
  http://localhost:5001/health
  
To shut down everything:

  docker compose down
